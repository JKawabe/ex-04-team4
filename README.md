# ex-04-team4

## 使い方
npm i  
npm run build  
npm start  
http://localhost:3010 にアクセス  

## 機能説明

1. ログイン・会員登録  
   ・管理者とそうでないアカウントでそれぞれログインできます。  
   ・ユーザー名、社員IDのどちらかとパスワードを入力することでログインできます。  
   ・「SIGN UP」で新規アカウントを作成することができます。  
   ・アカウント作成では英小文字2字・数字6文字からなる社員IDとユーザー名、パスワードを入力します。  
   ・社員ID、ユーザー名は他のユーザーと重複できません。  
   ・管理者アカウントは最初から用意されており、ユーザー名は「administrator」パスワードは「password」です。  
   ・セキュリティ上の観点から、他のユーザーを後述の方法で管理者に変更し、最初の管理者は削除してください。  
   ・「LOGOUT」でログイン画面に戻ります。  

2. スレッド  
   ・「START THREAD」からスレッドタイトルと概要を入力し、匿名か非匿名かを選択するとスレッドを作成できます。  
   ・作成したスレッドはスレッド一覧ページに表示され、クリックするとスレッドに入ることができます。  
   ・概要の編集ボタンを押すとスレッドの概要を変更できますが、タイトルは変更できません。  
   ・管理者の場合、スレッドページにスレッドの削除ボタンが表示され、スレッドを削除することができます。　　
   ・「お気に入り登録」を押すと開いているスレッドがお気に入りに登録され、「お気に入り解除」を押すと解除されます。  
   ・スレッド一覧ページの「お気に入りのスレッドを表示」を押すとお気に入りに登録したスレッドのみが表示されます。  
   ・「Meiji Forum」をクリックするとスレッド一覧画面に戻ります。

3. コメント 
   ・新規コメントは画面下側のコメント入力欄に入力できます。  
   ・既存のコメントに返信したい場合は、対象のコメントのreplyボタンを選択すると返信できます。  
   ・返信モードになっているかは入力欄の右のreply : を確認することで判断できます。  
   ・返信モードを取り消したい場合はreply番号をもう1度クリックすると解除できます。  
   ・返信機能のほかに前のコメントを引用することができます。これは返信と違って複数選択できます。  
   ・引用は>>(コメント番号)と入力すると引用されます。  
   ・引用をクリックするとその番号のコメントの位置に飛ぶことができます。  
   ・#(タグ)と打つとハッシュタグが生成されます。(空白までの文字列がハッシュタグのタグになります)  
   ・ハッシュタグをクリックするとそのハッシュタグを含むコメントのみが表示されます。  
   ・スレッドタイトルの文字をクリックするとハッシュタグのフィルタを解除することができます。  
   ・↑、↓のボタンを押すと画面の1番上側、下側に移動することができます。  
   ・「HOMEの戻る」または「Meiji Forum」をクリックするとスレッド選択画面に戻ります。

4. プロフィール  
   ・右上のユーザー名をクリックするとプロフィール画面に遷移します。  
   ・プロフィール画面ではテーマカラーとユーザー名を変更できます。  
   ・変更したいユーザー名またはテーマカラーを入力し、「変更を保存」を押すと変更できます。  
   ・管理者の場合は登録されているユーザーの一覧を閲覧することができます。  
   ・管理者はユーザーの権限変更とユーザーの削除を行うことが出来ます。
